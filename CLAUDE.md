# CLAUDE.md — portfolio

Portfolio pessoal em Next.js.

## Stack
- Next.js 14 (App Router)
- React 18
- Tailwind CSS
- TypeScript

## Comportamento Esperado
- Sempre usar TypeScript estrito
- Estilização via Tailwind, não CSS modules ou inline styles
- Componentes no App Router (Server Components por padrão, Client Components só quando necessário)
- Ao resolver um problema relevante, registrar em `obsidian-brain/snippets/` ou criar nota em `obsidian-brain/frontend/`
